科研小感-关于知识

发表于 2016-08-12 | 更新于 2016-11-17

感觉知识就像一张大的网。在做研究的时候，每当我想创造新知识的时候，即想从这张网的
某一个小区域出发往这张网的下面探索的时候，总是感觉被这一区域周围的网所限制，即被
那些有千丝万缕关系的知识，但是又不是直接相关的知识所牵绊。如果要把这部分知识也搞
懂的话，就又要花很多的力气。这种感觉，让我在搞科研的时候，感到十分的不舒服。到底
怎样才能找到一个比较舒服的状态？

家人的北京游

发表于 2016-06-05 | 更新于 2016-11-17

这一周两个姑妈带着奶奶来到北京游玩，这是奶奶第一次来北京。奶奶出生于1933年，今年
已经83岁了，身体和精神状态都非常的好。我之前只是知道奶奶已经过了80了，确切的生日
是我用她们的身份证买票的时候才知道的，做孙子的实在是过意不去。

老实说，已经很久没有逛过北京城里的景点了。不像刚来的两三年，每到放假或者清闲的周
末会和Sue妹妹计划着逛逛北京的各个地方，北海公园，什刹海公园，南锣鼓巷，美术博物
馆，恭王府。自从上了研究生之后，感觉生活的节奏明显不一样了，到了周末不是各个圈子
里约着吃饭，就是看电影。能在黑暗的电影院里坐下来安心的看两个小时电影，已经是很好
的享受了。
连故宫都是这个学期的春天才去的。

我的博客

发表于 2016-06-05 | 更新于 2016-11-17

一直想写博客很久了，主要有三个原因，一是因为平时在上网的时候看到好几个有意思的个
人博客，有学术的，有人文的（之后会慢慢推荐给大家），觉得自己也可以学习一下,搞一
搞；二是因为枯燥单调的博士生活，心情总是时不时的处于亚健康的状态，有时候触景生情
，有感而发，也想写上一篇发发牢骚；三是觉得自己写作能力本来就不强，由于搞了理科以
后尤其荒疏，每每想写点东西，满腔情感倒不出来，苦不堪言。但是由于本人实在是懒，懒
于记录，懒于折腾，所以博客一直没有开起来。
