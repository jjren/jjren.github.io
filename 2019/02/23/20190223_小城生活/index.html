小城生活

发表于 2019-02-23 | 更新于 2019-02-24

晚上和几个大学的好朋友在中关村吃完饭，一个人骑电动车回学校，骑到四环经过中关村的
那个大型十字路口，停下。看着各个方向的绿灯依次亮起，车如流水般穿行着，小汽车，公
交车，电动车，自行车。在这等待的短短五分钟里，突然有一种无助感、疲惫感、厌倦感。
眼前四周都是高大的天桥，比一般的过街天桥更高更大一些，像是一面墙，是不是太子李建
成当年在玄武门看着四周的高墙，也是这种感觉，渐渐繁荣的唐朝似乎明天再也和自己没关
系了。

记得9年前，刚刚来北京的时候，中关村应该给我这个来自小县城的少年上了大城市第一课
，电脑经销商一会花言巧语，一会威逼利诱，至今历历在目。虽然第一次来就被骗了一千块
，差不多是大半个月的生活费，但是当时好像也没怎么样。北京，对我来说，一切都是新的
，尤其是刚刚经过枯燥的高三，又经过艰难的病痛，我好像是脱笼的小鸟，新生一般。好像
心里面在说，北京，我来了，你看我来了，我说我会来的。

9年后的今天，对爱人亲人的思念，对毕业季的焦虑，对未来事业的未知，对未来生活的未
知，好像一张网，让我感觉在人生的路上往前走一步都气喘吁吁。确实过了飞扬的年少，工
作，薪资，买房，买车，结婚，生子，这些以前从来不会考虑的事情，现在的优先级在你考
虑的因素中越来越往前。有时候，不是我想成长，生活在推着我成长。学着接受一周七日的
宿舍，实验室两点一线；学着接受，食堂难言好吃，却几年不变的食物；学着接受老师给你
派发的每一项任务。

多久没有好好看一眼这个可爱的城市了，北京，在十字路口短短的一瞥，似乎已经陌生了。
9年的生活，似乎在这一刻比不上，9年前从火车站出来站在广场上看到你的样子，那样熟悉
。多久没有轻松的听一场喜欢或者谈不上喜欢的讲座，单纯只是为了接触一点新的有意思的
东西，在园子里待了9年后，似乎也没有那么熟悉了。清芬的水煎包，莲藕肉包，花生浆，
在溯溯毕业后再也没吃过了。草草的几口早饭，与其说是为了果腹，不如说是为了早餐这个
仪式。多久没好好的在图书馆看一会书了，多久没好好看园子里的四季，上一次在园子里乱
窜，拿着相机拍拍春景，不知道是哪年了。

是读博士吗？似乎是的，读博要求你五年如一日的坚守，好像一下午休息20分钟，也是对时
间的浪费。但又好像不是。读博并没有要求你放弃对生活的热爱。应该是年纪到了吧，在人
生的这个阶段里，脱去了稚气，过了强说愁的年纪，但是还没到成熟到笑对生活的年纪。就
像一颗青涩的果实，没了作为花朵时候的娇艳和花枝招展，但也没到秋天成熟时的硕果累累
。青涩的果实，只能努力的多给自己积累一些糖分。

想起春节里，在午后温暖的阳光里，和溯溯，爸爸，妈妈，在新小区的湖边散步聊天。和溯
溯，爸爸，奶奶，姑妈，在奶奶门前破旧的院子里，聊过去，折纸元宝。

小城，有新，有旧，在离开9年后的今夜，突然感觉分外的亲切，可爱，以及想念。
